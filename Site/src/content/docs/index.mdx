## CORE ARCHITECTURE

<CardGrid stagger>
  <Card title="Global Illumination" icon="sun">
    窓から差し込む一筋の光が床を叩き、柔らかな**バウンス光（間接光）**として部屋全体を包み込む「空気の重さ」を、独自のアルゴリズムで動的にシミュレート。
  </Card>
  <Card title="Physical Material (PBR)" icon="setting">
    木材の微細な吸光、大理石の鋭い鏡面反射、カーテンを透過する遮光。室内のあらゆる物質の質感を**物理ベース（PBR）**で正確に描き出します。
  </Card>
  <Card title="Sharp SSR / Reflection" icon="eye">
    静寂に満ちた室内の床やガラス。フレームレートを犠牲にすることなく、空間に圧倒的な深みと清涼感を与える**高精度なスクリーンスペース・リフレクション**。
  </Card>
  <Card title="Cinematic Color Grading" icon="document">
    建築雑誌のグラビアや映画の一コマのような、トーンマッピングとカラーグレーディングを実装。ポストプロセスによる極上の空気感。
  </Card>
