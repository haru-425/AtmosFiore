## プロフィール

ゲームプログラミング系の専門学校で3Dゲームエンジン開発およびレンダリングパイプラインを専攻。
DirectX 11 / C++を用いたローレベルな描画基盤の実装から、ゲームロジックへの応用までを一気通貫で開発できる強みを持っています。

チーム開発においては、GitHub Flowを用いたブランチ運用やCI/CDの自動化といった効率的なワークフローの構築を得意としています。

## スキルセット

### 言語

- **C++ (主要言語):** メモリ管理、ポインタ操作、オブジェクト指向設計、データ構造の最適化
- **HLSL:** PBR（物理ベースレンダリング）、各種ポストプロセスのシェーダー実装
- **C# / Java:** ツール開発、および外部サーバー連携用スクリプト

### グラフィックス API / エンジン

- **DirectX 11:** ゼロからの自作ゲームエンジン開発、定数バッファ・シェーダーリソース管理、パイプライン構築
- **Astro (Web):** ドキュメンテーションおよび高速な静的ポートフォリオ構築

### 開発環境・ツール

- **バージョン管理:** Git, GitHub (GitHub Flowによるチーム開発実践)
- **タスク・インフラ:** Slack/Discord連携自動化、Oracle Cloudを用いたマルチプラットフォームサーバー構築
